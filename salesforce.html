Salesforce is a special beast.


We dont' have a Salesforce form created.

We can test salesforce leads in two ways:

Using a VYSFID email query for the regullar player.

To do this:

  1. Ensure you have the Salesforce integration set up for your Org.
  2. Log into Salesforce.
  3. From the [Create New] dropdown menu, select [Lead].
  4. Complete the information for your New Lead and press the [Save] button at the top of the Lead Edit screen.
  5. The Lead page for your newly created lead will have a unique ID which you can copy from your browser's url bar.

    Example:
    ...salesforce.com/00Q46000005L6Q9

  6. Copy the ID and add it to the player share URL using the query string, vysfid

    Example:
    http://staging-qa.vidyard.com/share/3rcGzBYoLtyHL5aLM2LpSy?vysfid=00Q46000005L6Q9

Embedding the vysfid for the new, responsive player

The following page has a responsive player embedded into the code for the Salesforce contact 00Q46000005L6Q9.

  • Embedded responsive player with VYSFID for Salesforce. (Opens in new window)

If you want to test with your own contact, you will have to make a local copy of the html page and update the line containing, `data-vysfid=some_sf_id` to your specific ID. (see steps 1-5 above for details on creating/finding the ID.)
